Colorblind friendly figures of the paper related to `neutRal`:

Rocchini, D. (2026). Ecosystem shuffle: The `neutRal` R package to simulate species dance moves. Community Ecology.

Colorblind friendly figures can be attained by the [`cblindplot` R package](https://github.com/ducciorocchini/cblindplot)  

``` r
devtools::install_packages("ducciorocchini/cblindplot")

cblind.plot(figure1, cvd="deuteranopia") # example with deuteranopia
```
